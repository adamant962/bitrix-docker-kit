# Frontend assets

## CSS
- Sources: `local/templates/vega/css/`
- `style.css` — основной файл стилей
- `main.css` — дополнительные стили
- `font-rubik.css` — шрифт Rubik
- `fancybox.css` — Fancybox (модальные окна/лайтбокс)
- `animate.min.css` — CSS-анимации
- Bitrix overrides: нет отдельных `template_styles.css`

## JS
- `local/templates/vega/js/main.js` — core behavior
- `header.js` — функционал шапки
- `metric.js` — счётчики
- `marquee.js` — бегущая строка
- Нет npm / нет бандлера

## CDN dependencies
- jQuery 3.5.1
- Swiper (слайдеры)
- Slick (карусели)
- Fancybox (модальные окна, лайтбокс)
- Popper.js + Tippy.js (тултипы)
