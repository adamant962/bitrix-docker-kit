# Iblocks and web forms

Source: `local/php_interface/include/constants.php`

## Infoblocks
- `IBLOCK_ID_1C_PRODUCTS=48` — товары из 1С
- `IBLOCK_ID_SERVICES_1C=25` — услуги 1С
- `IBLOCK_ID_PRICE_SERVICES_1C=57` — цены на услуги 1С
- `IBLOCK_ID_NEWS=59` — новости
- `IBLOCK_ID_ARTICLES=9` — статьи
- `IBLOCK_ID_BLOG=65` — блог
- `IBLOCK_ID_AMENITIES=60` — удобства/возможности
- `IBLOCK_ID_BANNER_INDEX=27` — баннеры на главной
- `IBLOCK_ID_PROJECTS=30` — проекты
- `IBLOCK_ID_STOCKS=52` — акции
- `IBLOCK_ID_OUR_PROJECTS=56` — наши проекты
- `IBLOCK_ID_POKUPKA_I_LITSENZIROVANIE_SISTEM_1S=53` — покупка и лицензирование 1С
- `IBLOCK_ID_ESCORT_SERVICES_1C=63` — сопроводительные услуги 1С
- `HIGHLOADBLOCK_TYPE_ID=1` — HL-блок типов
- `HIGHLOADBLOCK_TYPE_SERVICES_ID=2` — HL-блок типов услуг

## Web forms
Bitrix24 формы интегрированы через `B24_FORM_ID_SCRIPT` (5 форм: click/139, click/191, click/150, click/149, inline/204). Стандартные веб-формы не используются.
